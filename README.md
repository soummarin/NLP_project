# NLP Text Summarization App

A Streamlit web application for text summarization using a fine-tuned BART model.

## Features
- Summarize text from uploaded PDFs or direct input
- View history of previous summaries
- Statistics dashboard with metrics and charts
- Download summaries as PDF reports

## How to Use
1. Upload a PDF file or enter text directly
2. Click "Summarize" to generate a summary
3. View results in the Home tab
4. Check History tab for past summaries
5. View Statistics for usage metrics

## Model
Uses a custom fine-tuned BART model for abstractive text summarization.
